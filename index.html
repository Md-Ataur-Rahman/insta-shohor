what is difference between local storage and session storage ?

Local Storage যে ওয়েব পেজেস সেটা tab মাধ্যম বাতিল করে| তাহলে Local Storage যে ডাটা গুলো আছে সেই ডাটা গুলো lost or Delete হয়ে যাই না| কিন্তু Session Storage যে ওয়েব পেজেস সেটা tab মাধ্যম বাতিল করে| তাহলে Session Storage যে ডাটা গুলো আছে সেই ডাটা গুলো সাময়িক ভাবে lost or Delete হয়ে যাই না|এতে বুঝা যায় যে লোকাল স্টোরেজ ডাটা স্টোরে থাকে কিন্তু সেশন স্টোরেজ ডাটা গুলো স্টোরে থাকে না|
